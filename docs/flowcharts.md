# Flowcharts – Mein Super-Start

## Kinderfluss
```
[Start] -> [Routine-Übersicht]
  -> {Tippe Aufgabe}
  -> [Timer läuft]
    -> {Timer endet?}
      -> Nein: [Motivation anzeigen]
      -> Ja: [Aufgabe abschließen]
        -> [Belohnung gutschreiben]
        -> {Noch Aufgaben?}
           -> Ja: [Nächste Aufgabe]
           -> Nein: [Tagesbelohnung auslösen]
```

## Elternfluss
```
[Login Eltern-PIN]
  -> [Aufgaben konfigurieren]
  -> [Timer & Reihenfolge festlegen]
  -> [Shop & Rewards verwalten]
  -> [Änderungen speichern]
  -> [Export / Import optional]
```
